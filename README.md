```bash

```

```bash
# Backend (.env)
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret

# Frontend (.env)
VITE_API_URL=http://localhost:5000
```

4. Run the application:
```bash
# Run backend (from backend directory)
npm start

# Run frontend (from frontend directory)
npm run dev
```
